# Lux Explorer Documentation

Welcome to the Lux Explorer documentation. This explorer is based on Blockscout and customized for the Lux Network ecosystem.

## Overview

The Lux Explorer provides:

- Block and transaction viewing
- Address tracking and analytics
- Smart contract verification
- Token tracking (ERC20, ERC721, ERC1155)
- API access for developers

## Documentation

- [Blockscout Alignment](/docs/blockscout-alignment) - How to stay aligned with upstream Blockscout
- [Multi-Chain Setup](/docs/multi-chain-setup) - Running explorers for multiple chains
- [Network Template](/docs/network-template) - Template for deploying network-specific explorers

## Quick Start

```bash
cd scripts
./run-mainnet.sh
```

Access the explorer at:
- Frontend: http://localhost:3000
- API: http://localhost:4000/api
